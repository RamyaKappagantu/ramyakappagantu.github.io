Projects

Fire Emergency Prediction Support Tool with the 911, City of Barrie

This project predicts the type and severity of the fire emergency using CAD and RMS historical fire emergencies data, weather and census data so that 911 can take necessary action and manage their resources. Performed ETL on the data sources using SSIS. The data has been cleaned by handling missing and inconsistent values, removing columns that are not relevant for further analysis. In addition, ArcGIS has been utilized to obtain dissemination areas of Barrie so that the fire emergencies are spotted according to the dissemination area. 3 different time-series forecasting models like LSTM (Long Short-Term Memory), VAR (Vector Autoregressive) and Prophet has been implemented to predict the next fire emergency. A web based dashboard has been built and deployed using Angular to visualize the fire emergency dissemination area, model metrics and the emergency severity(low, medium, high) using heatmaps. Metrics like accuracy, rmse has been calculated to evaluate the model performance and Hyperparameter tuning has been performed where model performance of LSTM has significantly improved by 6%. The conclusions drawn from this project helps the 911 to prioritize the emergencies, reduce casualties and plan resources more effectively.

Tech Stack: Python, SQL, SSIS, ETL, Time-Series Forecasting, Model Evaluation, Hyperparameter Tuning, Angular

Analysis of Public Drinking Water Fountains in Ottawa

This project involves collecting the data from Ottawa Open Datasets and performing data cleaning tasks like handling missing values, creating a new column to provide full address of the fountain, removing redundant columns (French translations), imputing values in CREATED_DATE column based on LAST_EDITED_DATE where a pattern of 2 to 3 secs difference has been noticed between the two columns. The Summary Statistics has been calculated using Sheets functions to understand typical values, outliers and quick insights. The data has been visualized in Tableau to answer key questions like "How many fountains are Open in Ottawa based on the duration of season?","How many fountains are Open Year Round based on whether they are present inside or outside the building?". All the insights are captured in a dashboard which also has filters so that users can get the corresponding visualizations. The dashboard also has an option to download as a pdf file which can be utilized like a report for presentations.
